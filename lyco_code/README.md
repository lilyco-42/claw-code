# lyco_code

`claw` CLI agent — cargo-binstall ready, built from [lilyco-42/claw-code](https://github.com/lilyco-42/claw-code).

## Install

```bash
cargo binstall lyco_code
```

This will download the prebuilt binary from GitHub Releases automatically.

## Manual Install

```bash
cargo install --git https://github.com/lilyco-42/claw-code lyco_code
```
